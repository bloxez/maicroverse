# Self-Contained Training Courses

This folder contains complete, self-contained training courses designed to teach specific topics and skills.

Each course includes:
- **Structured lessons** with clear learning objectives
- **Hands-on exercises** and practical examples
- **Project-based assignments** to reinforce concepts
- **All materials included** — no external dependencies required

These courses can be followed independently or combined to build a comprehensive learning path for mAIcro development and deployment.
