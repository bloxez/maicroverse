# Quick Start Projects

This folder contains quick start projects — completely functional, self-contained examples that demonstrate mAIcro capabilities and patterns.

Each project includes:
- **Pre-built application structure** ready to deploy
- **Sample data and workflows** to explore core features
- **Best practice implementations** for common use cases
- **Documentation** to customize and extend for your needs

Start with any project here to quickly get hands-on experience with mAIcro and build upon these foundations for your own applications.
